# 智造赋能科技提效大赛 - 项目方案陈述

---

## 一、基本信息

### 1. 战队名称
**智能流光**

### 2. 项目名称
**种草效能量化引擎-ImpactX**

### 3. 问题陈述
在当前营销环境下，企业面临广告投放效率低下和预算配置不合理的双重困境：**投放渠道多样但缺乏科学决策依据，预算分配凭经验主观判断，导致投入产出比难以量化和优化。**

### 4. 技术解决方案
基于**机器学习预测模型 + SHAP可解释AI技术 + 数学优化算法**，构建端到端的广告投放决策支持系统。通过随机森林算法学习历史投放数据中的规律，利用SHAP值量化各渠道对业绩的真实贡献度，结合约束优化算法自动生成预算分配建议方案，实现投放决策从"经验驱动"到"数据驱动"的转变。

### 5. 可衡量的价值成果
- **决策效率提升60%**：从传统的人工分析2-3天缩短至1小时内完成方案输出
- **预算利用率提升20-35%**：通过渠道贡献度量化和智能优化，优化预算配置
- **决策透明度100%**：每个渠道的调整建议都有SHAP值支撑，可清晰解释
- **适用场景广泛**：支持无约束探索、预算约束、渠道上下限等多种实际业务场景

## 二、项目详细方案

### 2.1 业务背景与痛点分析

#### 核心痛点
1. 品牌在小红书/抖音投入大量种草广告，但无法量化这些投入究竟为电商平台搜索和销量带来了多少贡献，从而无法科学设定种草预算。



---

### 2.2 技术架构与实现

#### 系统架构
```
数据层 → 模型层 → 分析层 → 优化层 → 展示层
```

#### 核心技术栈
- **前端框架**：Streamlit（快速构建交互式应用）
- **机器学习**：Scikit-learn Random Forest（预测模型）
- **可解释AI**：SHAP（Shapley Additive Explanations）
- **优化算法**：SciPy优化库 + 自适应约束处理
- **数据可视化**：Plotly（交互式图表）

#### 技术创新点

**1. SHAP值驱动的渠道贡献度量化**
- 传统方法依赖相关性分析，易受多重共线性影响
- SHAP值基于博弈论，能准确分解每个渠道的边际贡献
- 提供正负贡献方向，识别低效渠道

**2. 自适应约束优化策略**
```
智能约束模式：
- 无约束：探索理论最优解
- 智能约束：自动设置合理调整范围（±30%/±50%/±100%）
- 自定义约束：支持单渠道精细控制
```

**3. 投入产出鲁棒性验证**
- 计算边际效率指标（产出增长率/投放增长率）
- 自动识别不合理目标并预警
- 提供ROI和预算变化可视化

---

### 2.3 功能模块详解

#### 模块1：数据上传与预处理
- 支持CSV/Excel文件导入
- 自动识别数值型特征（包含逗号、百分号的数据清洗）
- 数据质量检测与缺失值处理

#### 模块2：模型训练与评估
- 随机森林回归模型（200棵决策树）
- 多维度模型评估指标：
  - R²（决定系数）：拟合优度
  - NRMSE（归一化均方根误差）：预测精度
  - 5折交叉验证：泛化能力
- 智能模型评分卡（0-100分）

**示例输出**：
```
训练 R²: 0.923    测试 R²: 0.887
训练 NRMSE: 0.045  测试 NRMSE: 0.068
交叉验证 R²: 0.891 ± 0.023
模型评分: 85分（良好）
```

#### 模块3：可视化分析
- **SHAP Summary Plot**：全局特征重要性排序
- **特征贡献度分析**：各渠道正负贡献分布
- **相关性热力图**：发现渠道间相互作用

#### 模块4：反推与预算优化
- **目标设定**：输入期望业绩目标
- **约束配置**：
  - 智能约束：自动生成合理调整范围
  - 自定义约束：单渠道上下限微调
  - 批量预设：锁定基准/仅增不减/灵活调整/历史范围
- **优化计算**：基于SHAP权重的约束优化
- **方案输出**：
  - 各渠道建议投放量
  - 预算变化与产出预测
  - ROI与边际效率分析
  - 约束状态检查（触限预警）

**实际应用场景**：
```
场景1：探索潜力
- 模式：无约束优化
- 输出：理论最优投放方案（可能大幅调整）

场景2：保守调整
- 模式：智能约束（±30%）
- 输出：稳健可执行方案

场景3：精细控制
- 模式：自定义约束
- 输出：符合实际业务限制的方案
```

### 2.4 实际应用流程

#### 典型使用流程
1. **上传数据**：导入历史投放数据（各渠道投放额 + 业绩指标）
2. **训练模型**：选择目标变量和特征，一键训练并查看评估结果
3. **分析贡献**：查看SHAP分析，识别高效/低效渠道
4. **优化预算**：设定目标和约束，生成建议方案

#### 决策示例
```
初始状态：
- 搜索广告：50万，SHAP权重：0.35
- 信息流广告：30万，SHAP权重：0.28
- KOL推广：20万，SHAP权重：0.18
- 基准GMV：1000万

目标：GMV提升至1100万（+10%）

系统建议（智能约束±50%）：
- 搜索广告：62万（+24%）← 高SHAP权重，增加投入
- 信息流广告：38万（+27%）← 高SHAP权重，增加投入
- KOL推广：15万（-25%）← 低SHAP权重，减少投入
- 预测GMV：1095万（达成率：99.5%）
- 总预算：115万（+15%），ROI：0.67

建议解读：
搜索和信息流是核心驱动力，增加投入能有效提升GMV；
KOL推广贡献度较低，可适当削减预算优化配置效率。
```

## 四、技术实现细节

### 4.1 模型训练逻辑
```python
# 核心流程
1. 数据预处理：特征标准化、缺失值处理
2. 训练集/测试集划分
3. 随机森林训练
4. 交叉验证评估泛化能力
5. 指标计算：R²、RMSE、NRMSE、CV-Score
```

### 4.2 SHAP值计算
```python
# 使用TreeExplainer加速计算
explainer = shap.TreeExplainer(model)
shap_values = explainer.shap_values(X_test)

# 特征重要性排序
feature_importance = np.abs(shap_values).mean(axis=0)
```

### 4.3 优化算法
```python
# 目标函数：最小化预测值与目标值的差距
minimize (model.predict(x) - y_target)²

# 约束条件：
- 预算约束：sum(x) ≤ total_budget
- 渠道约束：min_i ≤ x_i ≤ max_i
- 非负约束：x_i ≥ 0

# 求解器：SciPy SLSQP（序列最小二乘规划）
```

### 4.4 系统稳定性保障
- 数据变化自动检测（上传新数据时重置状态）
- 约束条件合法性校验（防止配置错误）
- 异常处理与友好提示
- 模型质量评分与预警
